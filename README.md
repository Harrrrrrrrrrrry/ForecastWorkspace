# PrismForecast

Website: https://prismforecast.vercel.app

A web-based forecasting project built for academic research and exploratory market analysis.

The application provides a simple interface for reviewing recent stock behavior, generating forecast-oriented outputs, and presenting results in a clear dashboard format.

## Project Scope

- Stock lookup and trend review
- Forecast dashboard
- Result summaries
- Reliability and limitation messaging

## Tech Stack

- Frontend: Next.js, React, TypeScript
- Backend: Python, FastAPI

## Status

This project is under active development as part of an academic research workflow.

This project is for educational and research purposes only. It is not financial advice or an investment recommendation.
